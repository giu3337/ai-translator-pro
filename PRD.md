* Objetivo: Traductor inteligente con contexto cultural + Chat interactivo.
* MVP (Mínimo Viable):
    * Input de texto -> Selección de idioma -> Output de traducción vía OpenAI.
    * Control de temperature (0 para precisión, 0.7 para fluidez natural).
    * Control de max_tokens para proteger tu billetera (Costo de Oportunidad).
* Stretch Goals:
    * Sistema de corrección: "IA, ¿por qué usaste esta palabra y no otra?".
    * Generación de imagen (DALL-E 3) basada en el contexto de la frase traducida.
    * Persistencia: Guardar el historial en base de datos.
