# AI Translator Pro 🌍⚡️

Traductor inteligente y premium que respeta contexto cultural, sensibilidad tonal y analiza gramática, potenciado por **Google Gemini 2.5 Flash** y diseñado en **Next.js 14+ con Tailwind 4 & Framer Motion**.

## 🚀 Características
- **Inteligencia Soberana**: Sobrescribe parámetros en tiempo real vía `generationConfig` (Temperature & Max Tokens).
- **Fricción Cero**: Atajos de teclado (`Cmd+Enter`) y portapapeles con confirmación visual.
- **Estética Pro**: Cristalomorfismo (`backdrop-blur-2xl`), Glows dinámicos, fuentes Inter y Mallas de Gradientes Animados (Mesh Gradients).

## 🛠 Instalación Local
1. `npm install`
2. Renombra `.env.example` o crea `.env.local` y añade tu `GOOGLE_GENERATIVE_AI_API_KEY=`
3. `npm run dev`
